## Terminal Deployment 

> **Note:**
> We require NodeJS V18 or higher

### Steps
1. Cloning
````bash
git clone https://github.com/polarisdevs/polaris-V1.git
cd polaris-V1
````

2. Installing
```bash
npm install
```

3. Run it!
```bash
npm start
```

4. Visit [localhost:8080](http://localhost:8080) for your site.
